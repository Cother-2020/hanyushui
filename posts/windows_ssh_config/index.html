Windows 下使用 SSH 密钥连接 Linux 服务器步骤#

步骤 1：生成 SSH 密钥对#

  1. 打开 PowerShell。

  2. 输入以下命令生成 RSA 类型的 SSH 密钥对：

    ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

  3. 按提示操作，默认保存路径为 C:\Users\<你的用户名>\.ssh\id_rsa。

  4. 可选择设置密码短语，增加安全性；也可以直接回车跳过。

步骤 2：复制公钥内容#

  1. 在 PowerShell 中输入以下命令查看并复制公钥内容：

    Get-Content $env:USERPROFILE\.ssh\id_rsa.pub

  2. 复制输出的整行内容（以 ssh-rsa 开头）。

步骤 3：将公钥添加到服务器#

方法一：使用密码登录服务器并添加公钥#

  1. 使用密码登录服务器：

    ssh your_username@server_ip

  2. 在服务器上执行以下命令：

    mkdir -p ~/.ssh
echo "粘贴你的公钥内容" >> ~/.ssh/authorized_keys
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys

方法二：使用 scp 命令上传公钥文件#

  1. 在本地将公钥文件上传到服务器：

    scp $env:USERPROFILE\.ssh\id_rsa.pub your_username@server_ip:~/

  2. 登录服务器并执行以下命令：

    mkdir -p ~/.ssh
cat ~/id_rsa.pub >> ~/.ssh/authorized_keys
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys
rm ~/id_rsa.pub

步骤 4：使用 SSH 密钥连接服务器#

  1. 在 PowerShell 中使用以下命令连接服务器：

    ssh your_username@server_ip

  2. 如果你使用了自定义私钥文件名，可指定路径：

    ssh -i "C:\Users\<你的用户名>\.ssh\your_private_key" your_username@server_ip

步骤 5：可选配置 - 简化 SSH 命令#

  1. 在 C:\Users\<你的用户名>\.ssh\ 目录下创建或编辑 config 文件。

  2. 添加以下内容：

    Host myserver
    HostName server_ip
    User your_username
    IdentityFile ~/.ssh/id_rsa

  3. 现在可以使用简化命令连接：

    ssh myserver

步骤 6：测试连接#

  1. 使用详细模式测试 SSH 连接：

    ssh -v your_username@server_ip

  2. 查看连接过程是否成功。

附注：#

  • 确保远程服务器上 sshd 服务正在运行。

  • 如果连接失败，请检查服务器的 /etc/ssh/sshd_config 中是否启用了以下配置：

    PubkeyAuthentication yes
AuthorizedKeysFile .ssh/authorized_keys

  • 修改配置后，重启 SSH 服务：

    sudo systemctl restart sshd
